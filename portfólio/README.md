# bea.tech — Portfólio de Beatriz Reis

Landing page profissional para divulgação de serviços digitais.

## Projetos exibidos
- **Rep.Rota** — Sistema de gestão de rotas para representantes e consultores
- **Finances Control** — Dashboard de organização financeira
- **Bot Assistente** — Secretaria virtual 24h no WhatsApp
- **Projeto Pharma** — Ferramenta clínica confidencial (NDA)

## Deploy no Vercel

```bash
npm i -g vercel
vercel login
vercel --prod
```

## Deploy no Netlify

```bash
npm i -g netlify-cli
netlify login
netlify deploy --prod --dir .
```

## Atualizar o site

Edite o arquivo `index.html` e faça o deploy novamente.
Para atualizações via Claude Code:
```bash
claude
```
